G.R.E.A.T¶

The Gang Resistance Education and Training (G.R.E.A.T.) program is a gang and delinquency prevention 13-lesson program delivered by law enforcement officers within a school setting. Developed locally in 1991 in Phoenix, the program quickly spread throughout the US.

Results from an earlier National Evaluation (1995-2001) found that the had an effect on several mediating variables associated with gang membership and delinquency but found no differences between G.R.E.A.T. and non-G.R.E.A.T. youths in terms of gang membership and involvement in delinquent behavior. Based in part on these findings, G.R.E.A.T. underwent a critical review that resulted in substantial program modifications.

In 2006, the National Institute of Justice awarded the University of Missouri-St. Louis funding to conduct a (second) National Evaluation of the G.R.E.A.T. program.

The Data¶

To assess program effectiveness, researchers conducted a RCT involving 3,820 students nested in 195 classrooms in 31 schools in 7 cities. A process evaluation consisted of multiple methods to assess program fidelity: 1) observations of G.R.E.A.T. Officer Trainings, 2) surveys and interviews of G.R.E.A.T.-trained officers and supervisors, 3) surveys of school personnel, and 4) "on-site," direct observations of officers delivering the G.R.E.A.T. program in the study sites.

Among US cities participating in G.R.E.A.T, the research team selected 7, varying in size, region, and level of gang activity, as primary target sites: Albuquerque, New Mexico; Chicago, Illinois; Greeley, Colorado; Nashville, Tennessee; Philadelphia, Pennsylvania; Portland, Oregon; and a city located in the Dallas-Fort Worth (DFW), Texas area.

Only the data from the student surveys, law enforcement officer surveys, and school personnel surveys are available. Dataset 1 (Student Survey) has 3,820 cases and 1,926 variables. Dataset 2 (Law Enforcement Survey) has 137 cases and 140 variables. Dataset 3 (School Personnel Survey) has 230 cases and 148 variables.

Variable Notation/Naming Conventions: V#1###2 where #1=Wave Number and ###2= Variable Number

The Study¶

The outcome evaluation employs an experimental longitudinal panel design (an RCT with long-term follow-up) where treatment is the G.R.E.A.T. program.

All students for whom active parental consent was obtained (3,820) were then asked to participate in the evaluation by completing a confidential group-administered pre-test questionnaire. Upon completion of the G.R.E.A.T. program in each school, students were then requested to complete post-tests and four annual follow-up surveys. Retention rates were excellent: 98.3% completed the pre-test, 94.6% completed the post-test, and 87.3% completed the one-year post-program survey.

The study:

  • Surveys students attending 31 public middle schools in the 7 cities mentioned above.
  • Uses the student responses to multiple waves of questionnaires (pre-test, post-test, and annual follow-up surveys in each of the following four years) to assess short- and long-term program effects, i.e. the extent to which students receiving G.R.E.A.T. differ from non-G.R.E.A.T. students in terms of their delinquent activity and gang involvement.
  • Additionally, examines the extent to which risk factors addressed in the G.R.E.A.T. program also differentiate the G.R.E.A.T. students from the control group.

G.R.E.A.T. has three stated goals: 1) to reduce gang membership, 2) to reduce delinquency, especially violent offending, and 3) to improve students' attitudes toward the police. The goal of the Esbenses et al. is to determine what effect, if any, the program had on students in terms of such 3 goals. In the study:

  • Gang membership is measured by a single-item question that is part of a larger set of questions about youth gangs: “Are you now in a gang?”.
  • To measure delinquency and violent offending, students completed a 15-item self-reported delinquency inventory treated as a composite measure of general delinquency, but also created a separate measure of violent offending consisting of three items (attacked someone with a weapon, used a weapon or force to get money or things from people, been involved in gang fights).
  • Improving relations with law enforcement, is measured by six questions tapping global attitudes to the police as well as two additional questions measuring students’ attitudes about police officers as teachers.

The analyses also included 26 proximal outcome measures (in addition to the five program outcomes discussed above) that tapped the extent to which the students enrolled in the G. R.E.A.T. program developed skills and attitudes that were promoted through- out the G.R.E.A.T. curriculum.

def setprev(df, col_lst): #### substitute NaN for previous value for each col listed
    for col in col_lst:
        for i in range(1, len(df)):
            if type(df[col].iloc[i])==float:
                df[col].iloc[i]= df[col].iloc[i-1]
    return df

#### Extract mean and SE from col 5:

def getSE(x): # this function is applied to each value of the column via map/apply
    if x and type(x) != float:
        if re.compile( "\((.*)\)" ).search(x):
            se =  re.compile( "\((.*)\)" ).search(x).group(1)
            se = se.strip(' ')
            if len(se) < 4:
                return float('0'+ se)
            else:
                return float(se)
        
def get_mean(x): # this function is applied to each value of the column via map/apply
    if x and type(x) != float:
        m = x.split('(')[0]
        m = m.strip(' ')
        if len(m) < 4:
            return float('0'+ m)
        else:
            try:
                return float(m)
            except:
                return m
            
def remove_char(x, ch):
    if x and type(x)!= float:
        return x.strip(ch)
    
def divdot(x, char, i): 
    if x and type(x)!= float and char in x:
        return  x.split(char)[i]
    
def tonum(df, colnum):
    for i, x in enumerate(df.iloc[: , colnum]):
        if type(x) != float and (len(x) > 5 or '(' in x):
            x = df.iloc[: , colnum+1][i]
        try:
            x = float(x)
        except:
            x=x     
    return df

def get_wave(x):
    if type(x) != float and 'V' in x:
        if 'V1' in x:
            return 'W1'
        elif 'V2' in x:
            return 'W2'
        elif 'V3' in x:
            return 'W3'
        elif 'V4' in x:
            return 'W4'
        elif 'V5' in x:
            return 'W5'
        else:
            return 'W6'
            
# Plot answer %
def plotvar(df, var, color='#3C9CC1'):
    df['summ'] = pd.to_numeric(df['summ'], errors='coerce')
    dfm = df[df['var_name'] == var].sort_values("summ")
    dfm.plot.barh(x='val', y='summ', title= dfm['lab'].iloc[0], color= color)
    
# Plot answer stats
def plotvar_stats(df, var_lst, stat, color='red'):
    df[stat] = pd.to_numeric(df[stat], errors='coerce')
    dfm = df[df['var_name'].isin(var_lst)]
    return dfm
